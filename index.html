Lagrange Interpolation

=> Let x0, x1, ..., xn be distinct points in the interval [a, b]
=> Suppose there's a function 'f' between the interval [a, b] that exists in the space of R
=> Suppose y0=f(x0), y1=f(x1), ..., yn=f(xn) are given
=> The main task of Lagrange Interpolation is to find a polynomial p(x) (in the space Pn) such that p(xi)=yi for i = 0, 1, ..., n

=> Let x0, x1, ..., xn be distinct points AND y0, y1, ..., yn be arbitrary values in Rn
=> There exists a unique polynomial p(x) in Pn such that p(x0), p(x1), ..., p(xn) exists for all cases of n
